# 🚛 Fleet Equipment Inventory Analysis (Excel Pivot Tables Project)

## 📌 Project Overview

This project analyzes fleet equipment inventory data using Microsoft Excel. The objective is to explore departmental equipment distribution through pivot tables, aggregation functions, and structured reporting techniques.

The analysis demonstrates how raw operational data can be transformed into meaningful insights for decision-making.

---

## 📁 Dataset Information

- Source File: Montgomery_Fleet_Equipment_Inventory_FA_PART_2_START.xlsx  
- Final File: Montgomery_Fleet_Equipment_Inventory_FA_PART_2_END.xlsx  

---

## 🛠 Tools & Techniques Used

- Microsoft Excel  
- Pivot Tables  
- AutoSum Functions (SUM, AVERAGE, MIN, MAX, COUNT)  
- Data Sorting & Filtering  
- Data Aggregation & Grouping
